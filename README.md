<h1 align="center">
CourseLink
</h1>

CourseLink is a collaborative platform designed for tutors to create, collaborate, and sell educational courses. The platform allows multiple tutors to contribute to courses, manage modules, and provides students with interactive learning tools, personalized certificates, and progress tracking.

Key Features

1. Collaborative Course Creation:
   - Tutors can create a course repository, add co-tutors, and collectively work on course modules.
  
2. Module Management with Approval:
   - Tutors can submit modules, and co-tutors review and approve content before merging it into the main course.

3. Student Enrollment and Progress Tracking:
   - Students can browse courses, enroll, and track their progress directly within their profile.

4. Secure Payment Integration:
   - Tutors can set pricing, and students can securely purchase courses with integrated payment gateways like Stripe.

5. Certification and Badges:
   - Tutors can customize certificates and badges for students, which are awarded upon course completion.

6. Interactive Features:
   - Comment sections within courses allow students to connect, discuss, and collaborate with each other.


Technology Stack

CourseLink is built on the MERN stack for a comprehensive and scalable platform:

- MongoDB: Stores course data, user profiles, progress tracking, and module versions.
- Express: Manages API requests and server-side logic, ensuring smooth interactions between users.
- React: Builds the responsive user interface, from the tutor dashboard to the student profiles.
- Node.js: Handles the backend server, processing requests and managing the collaborative features.


Installation Guide

To set up CourseLink locally, follow these steps:

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/isonikrish/CourseLink.git
   ```

2. Navigate to the project directory:

   ```bash
   cd CourseLink
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:5173` to start using CourseLink.
